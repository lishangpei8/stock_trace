GitHub 仓库配置
本应用将数据存储在你的 GitHub 仓库中，通过 GitHub Pages 提供访问，通过 GitHub Actions 自动更新行情。
首次部署指南
1
在 GitHub 创建一个新的公开仓库，将本项目代码推送上去
2
进入仓库 Settings → Pages → Source 选择 Deploy from a branch，Branch 选 main，文件夹选 / (root)，保存
3
进入 GitHub Settings → Developer settings → Fine-grained tokens，新建 Token，选择对应仓库，权限勾选 Contents (Read and write) 和 Actions (Write)
4
将仓库地址和 Token 填入上方，点击保存即可
